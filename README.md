- 1x **Automa Divider**

- 1x **Eggs Container**
- 1x **Food Token Lid**
- 1x **Food Tokes Container**
- 1x **Nectar & Dice Container**
- 1x **Nectar Lid**
- (2x **Lid**) or (2x **Lid alt**) or (1x **Lid** and 1x **Lid alt**)
